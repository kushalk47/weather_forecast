Weather Forecast API

Use this API to get weather data by station and date.

Format: http://127.0.0.1:5000/api/v1/stationid/date

Example: http://127.0.0.1:5000/api/v1/10/1987-12-28

Get all data for a station:

Example: http://127.0.0.1:5000/api/v1/10
